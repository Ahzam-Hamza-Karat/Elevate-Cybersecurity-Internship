## ðŸ“‹ Steps Followed
1. **Login** to Nessus Essentials web interface.
2. **Check IP Address** of the system to identify the network range.
3. Navigate to **My Scans** â†’ Click **New Scan** â†’ Select **Basic Network Scan**.
4. Configure:
   - **Name:** My Device Vulnerability Scan  
   - **Description:** Scanning the range of vulnerabilities  
   - **Target:** Local IP range (e.g., `192.168.253.0/24`)
5. **Save** the scan configuration.
6. **Launch** the scan and wait for completion.
7. **View Results** to check for detected vulnerabilities.
